# 3D-Light-Beam-Visualiser
This matlab app serves to act as a GUI for the development and visualisation of complex 3D light fields. It was developed as an extension tool to the pre-existing optical trapping tool box (OTT) matlab addon https://github.com/ilent2/ott. 

This project is built upon the work of software developed primarily by Issac Lenton seen [here](https://github.com/ilent2/ott) or directly as the matlab [link](https://au.mathworks.com/matlabcentral/fileexchange/73541-ott-optical-tweezers-toolbox). When combined with the optical trapping toolbox this matlab app acts as a GUI for the development, manipulation and visualisation of complex structured 3D light fields. Gaussian, Hermite-Gaussian, Laguerre-Gaussian and Ince-Gaussian beams can be layered in 3D space with full control of the polarisation, power, wavelength, numerical aperture, phase, etc. The beams can be manipulated in real time using a draggable interface I largely attribute to the work of [Prashanth Prakash](https://au.mathworks.com/matlabcentral/profile/authors/16177811) who's code made this element feasible. The interface also allows for a range of different visualisations of a given beam, exploring intensity of the electric field, radiance, phase etc. 

There are several updates I hope to make to when I have the time as well as some software bugs I'd like to iron out but in the meantime I hope this tool can still be of good use in teaching, research, recreation or study.
